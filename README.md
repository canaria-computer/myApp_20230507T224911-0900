# ニチケンテスター

## 背景

この ツールは 日本情報処理検定協会(以下、「**日検**」)の ホームページ作成検定 の自動採点をおこなうツールです。

日検 では 過去問と模範解答がソースコードとして公開されています。このツールを使うと模範解答と自分の解答を比較し
誤っている部分を検出して 採点を行い予想点数を算出します。

過去問による対策にお役立てください。
